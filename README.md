# 👨‍💻 My Contributions

As the primary developer of SafeSight, I was responsible for designing, developing, integrating, and refining both the AI-powered backend and the user-facing frontend. My contributions include:

## Frontend Development

- Designed and developed the complete user interface using **React.js** and **Vite**.
- Built a responsive and modern inspection dashboard for workplace safety monitoring.
- Implemented drag-and-drop image upload functionality using **React Dropzone**.
- Integrated the frontend with the Flask backend using **Axios**.
- Designed reusable UI components to improve maintainability and scalability.
- Enhanced the user experience by creating a clean, intuitive, and professional industrial-themed interface.
- Optimized the application layout by improving spacing, typography, visual hierarchy, and responsive behavior.
- Refined the inspection workflow to provide a seamless user experience from image upload to report generation.

---

## Backend Integration

- Integrated the React frontend with the Flask REST API.
- Implemented image upload and backend communication.
- Processed API responses and displayed detection statistics dynamically.
- Handled loading states, error handling, and successful inspection responses.
- Structured backend responses for efficient frontend rendering.

---

## AI Model Integration

- Integrated a custom-trained **YOLO** model for PPE detection.
- Connected the trained model with the Flask backend.
- Processed detection outputs to identify:
  - Safety Helmet
  - Safety Vest
  - Safety Gloves
  - Safety Boots
  - Safety Goggles
- Converted raw detection results into meaningful inspection metrics.

---
